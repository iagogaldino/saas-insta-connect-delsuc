# Instagram Connect - Frontend

Interface web do Instagram Connect para acompanhamento de métricas, conversas e sessões ativas.

## Como iniciar

1. Instale as dependências:

```bash
npm install
```

2. Crie o arquivo de ambiente com base no exemplo:

```bash
cp .env.example .env
```

3. Inicie o frontend:

```bash
npm run dev
```

## Observações

- O frontend depende do backend em execução.
- Ajuste o endereço do backend no arquivo `.env`, se necessário.
